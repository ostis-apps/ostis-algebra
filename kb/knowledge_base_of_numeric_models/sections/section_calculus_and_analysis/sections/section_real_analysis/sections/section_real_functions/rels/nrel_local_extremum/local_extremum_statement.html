Точка x0 называется точкой строгого локального максимума функции f(x), если существует такая окрестность этой точки, что для всех x из этой окрестности будет справедливо строгое неравенство : f(x) < f(x0).
Точка x0 называется точкой строгого локального минимума функции f(x), если существует такая окрестность этой точки, что для всех x из этой окрестности будет справедливо строгое неравенство : f(x) > f(x0).
