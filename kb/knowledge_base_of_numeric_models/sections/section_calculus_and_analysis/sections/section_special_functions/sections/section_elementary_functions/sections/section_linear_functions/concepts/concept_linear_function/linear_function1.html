Свойства:
  • k (угловой коэффициент прямой) является тангенсом угла α∈(α [0; π/2)∪(π/2; π)), который прямая образует с положительным направлением оси абсцисс.
  • При k>0, прямая образует острый угол с осью абсцисс.
  • При k<0, прямая образует тупой угол с осью абсцисс.
  • При k=0, прямая параллельна оси абсцисс.
