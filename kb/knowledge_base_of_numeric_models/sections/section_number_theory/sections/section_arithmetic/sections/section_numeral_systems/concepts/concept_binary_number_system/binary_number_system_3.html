Преобразование двоичных чисел в десятичные. Допустим, дано двоичное число 1100012. Для перевода в десятичное запишите его как сумму по разрядам следующим образом:
  • 1*25+1*24+0*23+0*22+0*21+1*20=49
    • То же самое чуть иначе:
  • 1*32+1*16+0*8+0*4+0*2+1*1=49
    • Сложите получившиеся десятичные числа. Таким образом, двоичное число 1100012 равнозначно десятичному 4910.
