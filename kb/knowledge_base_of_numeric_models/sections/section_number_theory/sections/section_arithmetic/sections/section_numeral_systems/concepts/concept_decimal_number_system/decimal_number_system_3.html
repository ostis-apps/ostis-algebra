• Разложение натурального числа по разрядам: любое натуральное число а, записанное в десятичной системе счисления с помощью n цифр (n-разрядное число) равно сумме числовых значений данных цифр в позиционной записи этого числа:
  • а=аn-1...a1a0=аn-1*10n-1+аn-2*10n-2+...+а1*10+а0
    • где a0,a1,...,аn-1 – цифры от 0 до 9
